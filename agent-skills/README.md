# Agent Skills

This directory follows the Agent Skills format from [agentskills.io](https://agentskills.io).

Each subdirectory is a standalone skill with a required `SKILL.md` containing:

- YAML frontmatter (`name`, `description`)
- Markdown instructions

Available skills:

- `fast-local-validation`
- `component-generation-flow`
- `storybook-playground-parity`
- `safe-generated-file-updates`
- `framework-parity-checks`
- `run-agent-jobs`
