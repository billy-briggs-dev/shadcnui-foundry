# @shadcnui-foundry/playground-angular

Static Angular playground for previewing generated component output.

## Commands

- `pnpm --filter @shadcnui-foundry/playground-angular build` builds to `dist/index.html`
- `pnpm --filter @shadcnui-foundry/playground-angular dev` serves the built playground locally

The page renders shell previews and source snippets for generated:

- `Button`
- `Input`
- `Card`
